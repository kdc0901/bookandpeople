# Book and People Website

도서 관리 및 사용자 인증 기능을 갖춘 웹 애플리케이션입니다.

## 주요 기능

- 사용자 인증 (회원가입, 로그인)
- 도서 관리 (CRUD)
- 관리자 기능
- 반응형 디자인

## 기술 스택

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB
- Template Engine: EJS

## 설치 및 실행

1. 의존성 설치:
```bash
npm install
```

2. 환경 변수 설정:
- `.env` 파일 생성
- 필요한 환경 변수 설정

3. 서버 실행:
```bash
npm run dev
```

## 프로젝트 구조

```
my-website/
├── public/          # 정적 파일
├── src/            # 서버 코드
├── views/          # EJS 템플릿
└── package.json    # 프로젝트 설정
```
